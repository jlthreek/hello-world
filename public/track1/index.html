Track 1 공식 과제

유무인 복합체계(MUM-T)와 Counter-UAS 분야의 공개 도전 과제 6선. 각 과제는 배경 · 과제 정의 · 세부 요구사항으로 구성됩니다.

#1범용
1인 다중 무인기 동시 통제
Multi-UxV Control

// 배경

최근 전장에서 드론을 비롯한 UGV, USV 등 다양한 무인 시스템의 운용 비중이 급증하고 있습니다. 그러나 현행 운용 방식은 장비 증가에 비례해 운용 인력 또한 동반 요구되는 선형적 구조 한계를 지니고 있습니다. 이는 인구 절벽 및 병력 감소 직면국(한국 및 APAC)의 장기 전력 운용 측면에서 지속 불가능합니다. 따라서 미래 전장은 단일 운용자가 다수의 무인체계를 통합 감독·제어하는 유무인 복합 체계로의 전환이 필수적입니다.

// 과제

단일 운용자가 공중·지상·해상의 이종(UxV) 무인체계 3대 이상을 동시에 임무 부여, 감시, 통제할 수 있는 체계를 개발하라.

운용자 인지부하(Cognitive Load)를 최소화하면서도 임무 성공률을 유지하거나 향상시킬 수 있는 방안을 제시할 것.

// 세부 요구사항

  • 임무 단위 추상화(개별 조종 → 임무 의도 입력) 인터페이스
  • 다수 무인기의 자율 임무 분산·역할 할당
  • 운용자 인지부하 모니터링 및 우선순위 알림
  • 이상·위협 발생 시 사람 개입 지점(human-in-the-loop) 설계
  • 인력 절감률·임무 성공률 측정 프레임워크
#2특수상황
산악·도심 혼재 지형 자율 드론 운용
Terrain-Aware Autonomy

// 배경

최근 전장에서 드론은 핵심 전력으로 자리잡았지만, 대부분의 자율비행 알고리즘은 공중 최단거리 이동 중심의 단조로운 지형(예: 우크라이나)을 전제로 설계되어 있어, 복합 지형 환경을 반영한 고도화가 요구됩니다. 반면 APAC 작전 환경은 산악, 도서, 해안선 및 고밀도 도심이 중첩된 복합 지형의 특성을 보입니다. 이로 인해 작전 시 GPS 음영, 통신 단절, 전자전(EW) 저해 행위, 지형적 차폐물 등의 고위험 변수가 복합적으로 작용하여 단순 선형 경로 비행을 통한 임무 달성이 제한됩니다. 이에 따라 차세대 자율 시스템은 단순 목적지 이동 차원을 넘어, 복잡 지형과 동적 위협을 실시간으로 인지·판단하고 자체 생존성을 극대화할 수 있는 '지형 인지형 자율성(Terrain-Aware Autonomy)' 확보가 시급합니다.

// 과제

산악·도서·도심이 혼재된 APAC 환경에서 드론이 지형을 활용해 이동하고, 통신을 유지하며, 전자전 위협 속에서도 임무를 지속할 수 있는 Terrain-Aware Autonomy 체계를 개발하라.

// 세부 요구사항

  • 산악·도서 지형을 고려한 3D 경로 계획 알고리즘
  • 지형을 엄폐물로 활용하는 Terrain Following / Masking 비행
  • 전자전(EW) 신호의 탐지·방향 추정 및 회피 기동
  • RF·LTE·위성통신 품질을 예측해 통신 단절을 최소화하는 경로 설계
  • 위협 발생 시 실시간 자율 임무 재계획(Autonomous Re-Planning)
  • 다수 드론의 협업 비행 및 자율 임무 분산
#3특수상황
수도권 영공 소형드론·풍선 침투 대응
Gray-zone C-UAS

// 배경

최근 소형 무인기 및 침투용 풍선 도발 사례는 수도권 영공이 더 이상 전통적인 항공 위협에만 국한되지 않는 저강도 회색지대(Gray-zone) 도발 공간으로 변모했음을 시사합니다. 특히 LSS(저고도·저속·저신호) 비행체는 기존 광역 레이더 체계로 탐지·식별이 제한되며, 인구 밀집 도심지 특성상 물리적 타격 시 심각한 부수적 피해(Collateral Damage)를 초래할 수 있습니다. 따라서 단순 탐지 성능의 고도화를 넘어, 극히 제한된 골든타임 내에 위협 수준을 정확히 식별·평가하고 최적의 저위험 대응 옵션을 결심할 수 있는 지능형 지휘통제 능력이 요구됩니다.

// 과제

도심 환경에서 소형 드론·풍선 등 저신호 비행체를 탐지, 추적, 식별하고 위협도를 평가하여 대응 결심을 지원하는 Gray-zone Counter-UAS 체계를 개발하라.

(인구 밀집 도심 상공에서 저고도 소형 비행체(드론·풍선)를 탐지→추적→식별→대응 결심까지 지원하는 의사결정 체계)

// 세부 요구사항

  • RF·EO/IR·레이더 융합 저신호 표적 탐지
  • 도심 클러터(잡신호) 환경에서의 오탐 저감
  • 비행체 유형(드론/풍선/조류) 분류 및 위협도 평가
  • 부수피해를 고려한 대응 옵션(소프트킬/하드킬) 추천
  • 다중 침투체 동시 추적 및 경보 시각화
#4특수상황·공중 (MUM-T)
CCA 임무배분 샌드박스
CCA Mission Allocation Sandbox

// 배경

미래 공중전은 유인기 1대가 다수의 CCA(협업 전투기)·무인기를 거느리는 유무인복합(MUM-T)으로 전환되고 있다. 조종사 1명이 ISR·전자전·기만·센서확장 등 서로 다른 임무의 무인 편대를 동시에 지휘해야 하므로, 임무를 자동 배분해 인지부하를 줄이는 체계가 필요하다. (T1 '1인 다중 무인기 통제'의 공중 CCA 버전)

// 과제

유인기 1대 + CCA/무인기 2~4대를 가정해 ISR·EW·Decoy·Sensor Extension 등 임무를 상황에 맞게 자동 배분하는 유무인복합 시뮬레이터를 개발하라. (공개·가상 데이터만)

// 세부 요구사항

  • 임무 유형(ISR/EW/Decoy/센서확장)별 자동 배분 로직
  • 위협·자산 상태 변화 시 실시간 재배분
  • 조종사 인지부하 최소화 인터페이스
  • 가상 시나리오 기반 임무 성공률 평가
  • 유인-무인 역할 분담·교전규칙 반영
#5Mini Operational Test
자율 요격 드론 Mini Operational Test
Autonomous Counter-UAS Interceptor

// 배경

최근 전장에서는 FPV 드론을 이용한 직접 타격에서 나아가, 다수의 소형 드론이 동시에 침투하는 군집형 위협(Swarming Threat)이 빠르게 증가하고 있습니다. 이에 따라 Counter-UAS 또한 단순 수동 조종을 넘어, 다수의 요격 드론 운용과 AI 기반 자율 종말 유도(Autonomous Terminal Guidance)를 포함하는 방향으로 발전하고 있습니다. 실제 운용에서는 운용자가 목표 인근까지 접근시키고, 마지막 구간은 드론이 스스로 표적을 탐지·추적하여 대응하는 Human-on-the-loop 운용 개념이 중요해지고 있습니다. 이번 과제는 이러한 Counter-UAS 발전 단계를 실제 실내 환경에서 안전하게 축소 모사(Mini Operational Test)하여 검증하는 것을 목표로 합니다.

// 과제

실제 Counter-UAS 운용 절차를 실내 환경에서 축소 구현한 Mini Operational Test를 수행하라. 참가팀은 아래 3가지 운용 미션 중 하나 이상을 구현하여 Counter-UAS 체계를 제안한다.

Test 1 — Manual FPV Intercept

FPV 방식으로 요격 드론을 직접 조종하여 목표(Ring)를 통과하여 기본 요격 능력을 입증한다.

Test 2 — Multi-Interceptor Response

최소 2대 이상의 요격 드론을 활용하여 다중 위협 상황을 대응한다. 운용 방식(지상 Launch, 공중 Launch, 릴레이 등)은 자유이며, 각 요격 드론이 순차적으로 목표(Ring)를 통과하면 성공으로 인정한다.

Test 3 — Human-on-the-loop Autonomous Intercept

운용자가 GCS를 이용해 요격 드론을 목표 인근(약 5~15m)까지 접근시킨 뒤 자율 모드로 전환한다. 이후 드론은 스스로 표적을 탐지·추적하여 목표(Ring)를 통과해야 한다.

// 세부 요구사항

  • FPV 기반 Manual Intercept 운용
  • Vision 기반 표적 탐지(Object Detection)
  • 자율 표적 추적(Target Tracking)
  • Human-on-the-loop 기반 자율 종말 유도
  • 다수 요격 드론 협업(Multi-Interceptor Coordination)
  • Launch 방식(지상·공중·릴레이) 설계
  • 운용 개념(CONOPS) 및 Counter-UAS 아키텍처 제안
  • AI 기반 자율 요격 알고리즘 및 시스템 확장성
#6범용
무선침묵 환경 피아식별 (IFF)
Radio-silent IFF

// 배경

유무인 복합체계(MUM-T) 환경에서 아군 드론을 운용함과 동시에 적 드론을 무력화하기 위해서는, 전장의 인간 군인과 AI 기반의 무인 체계(기계)가 통신 노출 없이 상호간에 '무엇이 아군이고 무엇이 적인가'를 실시간으로 정확히 식별할 수 있어야 합니다. 그러나 기존의 IFF 체계는 지속적인 신호 방출로 인해 아군의 위치를 노출시키거나, 중간자(MITM) 및 재전송(Replay) 공격에 취약하여 적 드론이 아군으로 위장할 위험이 존재합니다. 따라서 무선 침묵 상태에서도 인간과 기계가 서로를 은밀하게 알아볼 수 있는 피아식별 기술이 필수적이며, 이는 무인 체계의 자율 운용(Autonomy)과 대드론 대응(Counter-UAS) 영역을 유기적으로 연결하는 핵심 고리입니다.

// 과제

전파 방출을 최소화한 채 아군·적 무인기를 식별할 수 있는 암호학적 2-컴포넌트 피아식별 체계를 개발하라.

중간자(MITM)·재전송(replay) 공격에 견딜 수 있는 방향 제시.

// 세부 요구사항

  • 무선침묵(저방출) 기반 피아식별 프로토콜
  • 암호학적 2-컴포넌트 challenge-response 설계
  • MITM·replay 공격 방어 메커니즘
  • 아군 드론 운용체계와 대드론 체계 간 식별 정보 공유
  • 식별 실패 시 fail-safe 교전규칙 연동
